# PRO-CTR

## COC-hoofd regels. 
Volg de CoC. Als een COC regel wordt verbroken zonder geldige reden krijg je een strike, bij de derde strike ben je eruit (elke strike gesprek met docent) 


### Sociaal: 
Behandel elkaar met respect. Je spreekt elkaar altijd aan op onwenselijk/afleidend gedrag. Niet te veel met andere groepen bemoeien. Zorg dat je niet wordt afgeleid door dingen als sociaal-media, Netflix en games. Je bent op tijd aanwezig op school, zo niet bespreek dit dan een half uur van tevoren. Wanneer je een taak aanneemt rond deze optijd af, zo niet communiceer dit tijdig. Kritiek en feedback richt je op het product/werk en niet op de persoon. Jij weet waar iedereen (binnen jouw team) mee bezig is. 

### Code:
Voor een push naar dev test de game in unity. Eerst merge dev naar jou voordat jij naar dev merged. Gebruik AI alleen om informatie te verzamelen of korte stukjes code. Geen code van andere aanpassen zonder toestemming. Geef elkaar de kans om te groeien. Wanneer je vastloopt, vraag om hulp. 

### Andere regels:
Wanneer mogelijk, zorg ervoor dat iedereen toegang heeft tot de bestanden. Als je niet weet wat je kunt doen laat je dat het groepje weten. Als nodig, kan je een korte pauze nemen, eigen verantwoordelijkheid Je bent verantwoordelijk voor je eigen werk Userstories bevatten een definition of done Daily standups worden in het eerste halfuur van de dag in het stamgroep kanaal gezet Aan het einde van de week wordt er een
retrospective gehouden waarbij besproken wordt wat er goed ging en wat er beter kan gaan.
